Predictions

We use state-of-the-art machine learning algorithms to predict the winner of football games each week. Our bots go head to head each week, constantly competing to be the best predictor.

About Our Bots

Each "bot" is actually a different machine learning algorithm! As the weeks go by, these algorithms will become "smarter" as they learn more and more each week. Some of the algorithms include:

  • K Nearest Neighbor
  • Random Forest
  • Logistic Regression
  • XGBoost
  • Linear and RBF Support Vector Machine
  • & More...
