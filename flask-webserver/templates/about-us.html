About Us

I am a Cybersecurity student who wanted to learn more about programming and machine learning. I partake in a football "pickem" league every year, and I attempted to create something that could pick better than me. To share the results of my project, I created this website.

How did you do all of this?

When I started these predictions in 2018, the only relevant knowledge that I had was a little bit of python. I could write a few scripts that could do basic things, but not much more. When I started this project, I was in over my head. It took a lot of time, dedication, and effort to get this to where it is today.

The best advice I can give is to continuously reseach, and never stop learning. I never thought that I would be able to actually run these predictions, and now I'm running a website! (I built the website with python as well, it's running with Flask!)
